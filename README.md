# Primer Dibujo en SFML :D

## Ejecución y Requirements
Para poder emplear el programa, es necesario contar con CMake version 3.28 (mínimo), además de contar con SFML 3.

Para compilar el programa es necesario ejecutar (en el directorio raiz del proyecto)

   ```
   cmake -B build
   cmake --build build
   ```
Para correr el programa:
   ```
   ./build/curva
   ```

## Recursos adicionales
Para esta tarea, se utilizó la herramienta de Geogebra que construye las curvas de Bézier mediante puntos proporcionados en el plano.
