# Primer Dibujo en SFML :D
Dibujamos una imagen a partir del algoritmo de Chainkin.

## Contenido
Es un programa bastante sencillo, consta de dos directorios llamados include y source, que contienen "Vector2.h" y "main.cpp" respectivmente, mientras que Vector2.h contiene una implementación de vectores en dos dimensiones, main.cpp se encarga de dibujar la imagen solicitada.

## Ejecución y Requirements
Para poder emplear el programa, es necesario contar con CMake version 3.28 (mínimo), además de contar con SFML 3.

Para compilar el programa es necesario ejecutar (en el directorio raiz del proyecto)

   ```
   cmake -B build
   cmake --build build
   ```
Para correr el programa:
   ```
   ./build/curva
   ```
